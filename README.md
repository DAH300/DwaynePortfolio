# Dwayne Howell - Portfolio

Modern, responsive front-end portfolio website built with **Next.js** and **Docker**.

## Live Site
[https://dhowell.netlify.app/](https://dhowell.netlify.app/)

## Features
- Clean, modern UI/UX design with excellent mobile responsiveness
- Fast performance using Next.js App Router and server-side rendering
- Dockerized setup for consistent development and production environments
- CI/CD ready configuration
- Showcasing front-end development, UI/UX principles, and full-stack skills

## Tech Stack
- **Framework**: Next.js (React)
- **Styling**: Tailwind CSS
- **Containerization**: Docker + Docker Compose
- **Version Control**: Git + GitHub

## Running Locally

### With Docker (Recommended)
```bash
docker compose up --build
```
The site will be available at http://localhost:3000
Without Docker

```bash
Bashnpm install
npm run dev
```
